# bv-deeponet-gp-ctl
BV-DeepONet with GP-CTL: a 3D operator learning framework for robust structural prediction across 18 topologies. Uses binary voxelization to ensure density invariance, dual-metric active transfer to solve data scarcity, and 4Q-HEM to suppress high-error anomalies. Improves R² to 0.9681, cuts sample needs by 66%, and speeds up convergence by 90%.
